# b00p
## Problem Statement

boop.

## Solution

Every other character is input in reverse. To get the flag, read every other character backwards.

| f | } | a | p | { | 0 | l | p | 0 | 0 | w | f | 0 | g | y | l |
|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| f | l | a | g | { | f | l | 0 | 0 | p | w | 0 | 0 | p | y | } |

Flag: `flag{fl00pw00py}`
